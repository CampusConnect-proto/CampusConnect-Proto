# **App Name**: Campus Connect

## Core Features:

- User Authentication: Secure user authentication and role management (Student/Owner) using Firebase Authentication with email/password and Google OAuth.
- Property Listing: Owners can list properties with details such as rent, capacity, facilities, images, and contact information.
- Advanced Search & Filtering: Students can search and filter properties based on location, max rent, facilities, distance from college, and room type.
- Property Details View: Detailed view of properties with metadata, image gallery, and owner contact information.
- Review and Rating System: Students can submit ratings and reviews for properties, with average ratings displayed.
- Targeted recommendations: Generative AI tool which uses the location to give students targeted property recommendations based on location and other key factors

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke trust and calmness, reflecting the assurance of finding ideal housing.
- Background color: Light grey (#F5F5F5), a desaturated version of the primary, for a clean and neutral backdrop.
- Accent color: Pale violet (#9575CD), analogous to blue, adding a touch of creativity and modern feel, different enough in saturation and brightness to make interactive elements stand out.
- Body and headline font: 'Arial', a grotesque-style sans-serif, known for a modern, neutral aesthetic; suitable for a clean and user-friendly experience
- Use clear and consistent icons from Material UI to represent different property features and amenities.
- Clean and intuitive layout with a focus on usability and easy navigation.
- Subtle animations and transitions to enhance user experience, such as property loading and filtering.
